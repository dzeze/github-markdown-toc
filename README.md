# GitHub-Markdown-Toc

## About

This is a chrome extension that supports parsing articles on GitHub and generating a sidebar directory for indexing of article titles.

> 这是一款 chrome 扩展，支持对 GitHub 上的文章进行解析，生成侧边栏目录，方便进行文章标题进行索引。
